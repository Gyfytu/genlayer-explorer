# GenLayer Hub 🧠

A sleek, cyberpunk-themed web hub for the **GenLayer** ecosystem — featuring live network stats, an AI-powered explainer, and an events dashboard.

## Features

- **Terminal Hero** — Animated typing terminal with GenLayer branding
- **AI Explainer** — Simulated Intelligent Contract that breaks down GenLayer in simple terms
- **Events Dashboard** — Browse upcoming GenLayer community events
- **Live Network Pulse** — Real-time ticker showing validator count, contracts executed, and consensus time
- **Admin Panel** — Hidden admin dashboard for managing events

## Tech Stack

- **React** + **TypeScript**
- **Vite** — Lightning-fast dev server & build
- **Tailwind CSS** — Utility-first styling with custom design tokens
- **shadcn/ui** — Accessible component primitives
- **Framer Motion** — Smooth animations
- **Lucide React** — Icons

## Getting Started

```sh
# Clone the repo
git clone <YOUR_GIT_URL>
cd genlayer-hub

# Install dependencies
npm install

# Start dev server
npm run dev
```
